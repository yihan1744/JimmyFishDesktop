🍎 macOS

ZIP · Universal (Intel & Apple chip) · ~65 MB

Download for Mac

After downloading 下载完成后

1. Unzip, then double‑click the game to run.

1. 解压 ZIP，双击游戏运行。

2. If you see “Not Opened” (Done / Move to Bin): click Done — not Move to Bin. Then use Open Anyway in Settings (below). Once is enough; after that it opens normally.

2. 若出现「未能打开」（「好」 / 「移到废纸篓」）：点「好」，勿点「移到废纸篓」；再到系统设置中使用「仍要打开」（见下）。通常只需一次，之后即可正常打开。

Open Anyway — steps

  1. System Settings → Privacy & Security
  2. Under Security, find the blocked app message → Open Anyway (password if asked).
  3. Open the game again → confirm Open.

仍要打开 — 步骤

  1. 系统设置 → 隐私与安全性
  2. 在安全性中找到拦截提示 → 仍要打开（按需输入密码）
  3. 再次打开游戏 → 选打开
