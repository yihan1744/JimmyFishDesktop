# JimmyFishDesktop (Godot)

1. Install **Godot 4.6** (or the version in `config/features` inside `project.godot`).
2. In Godot: **Project → Open** and select **this folder** (the one that contains `project.godot`).

- **Build / export:** see `docs/build.md`.
- **Blender → Godot** (glTF, checks, baking UVs): see `docs/pipeline/`.
- **Python tools** (run with Blender in batch mode): `tools/`.
